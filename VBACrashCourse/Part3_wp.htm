Visual Basic supports the usual range of program flow statements, including If, For, and While. This chapter covers the branch statements, and the next one covers loops.

If-Then-Else

The basic If statement is coded as follows:

    If CustomerName = "Hughes" Then
        Debug.Print "Customer: " & CustomerName
    End If
    
    If i <> 1 Then
        Debug.Print "i does not equal 1"
    Else
        Debug.Print "i equals 1"
    End If

There is also a shortened If syntax for when there is only one statement to execute.

    If i = 0 Then i = 1

The executed statement must be on the same line as the If keyword

Select Case Statements

When there are a large number of possibilities that you wish to test for, a series of If statements may be quite cumbersome, so VB provides a Select Case statement to use instead: (this is equivalent to switch in C++).

    Select Case i
    Case 1:
        Debug.Print "One"
    Case 2:
        Debug.Print "Two"
    Case Else:
        Debug.Print "Some other number"
    End Select
